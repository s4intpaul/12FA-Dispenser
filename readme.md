This is the web app repo for the 12FA Soap Dispenser, designed by saintpaul, honorary member
of UC Robo team for the Terrible Ideas 2026 hackathon. This web app/repo was designed entirely 
by me for the purposes of this competition. 

It is designed to interface with the Raspberry Pi touchscreen/display module and 
serve as the first point-of-contact in the UX journey, when a user first interacts 
with the soap dispenser. 

the stack includes: 

- ngrok for the local deveoplent server-- instead of using my own ip.
- google auth/requests library for time-based-one-time-password (TOTP) feature.
- linkedin API/developer setting for connectivity.
- html/css for front end.

More libraries will be added later on.
