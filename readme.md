This is the web app repo for the 12FA Soap Dispenser, designed by UC Robo team for 
the Terrible Ideas 2026 hackathon. This web app/repo was designed entirely by me for
the purposes of this competition. 

It is designed to interface with the Raspberry Pi touchscreen/display module and 
serve as the first point-of-contact in the UX journey, when a user first interacts 
with the soap dispenser.
